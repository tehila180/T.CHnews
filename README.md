# 📰 T.CHnews – Community News Platform

מערכת חדשות וקהילה מתקדמת ל־Web ול־Mobile, המאפשרת למשתמשים לפרסם פוסטים, להגיב בזמן אמת, לנהל פרופיל אישי ולצרוך חדשות חיצוניות.  
המערכת נבנתה כ־Full Stack Application עם דגש על אבטחה, הרשאות, חוויית משתמש ועיצוב מודרני.

---

## 📌 תיאור כללי

ה T.CHnews היא פלטפורמה קהילתית מודרנית המשלבת:
- מערכת פוסטים ותגובות בזמן אמת
- ניהול משתמשים והרשאות מתקדם
- פרופילים אישיים
- פאנל ניהול (ADMIN)
- חיבור ל־API חיצוני של חדשות
- התאמה מלאה לווב ולמובייל מאותו קוד

המערכת תוכננה להיות מאובטחת, נוחה לשימוש, קלה לתחזוקה וסקיילבילית.

## 🧠 ארכיטקטורת המערכת

המערכת בנויה בארכיטקטורת Client → Cloud (BaaS), ללא שרת ביניים.

### Frontend
- React Native (Expo)
- TypeScript
- אפליקציה אחת המותאמת ל־Web ול־Mobile
- ניווט בין מסכים
- UI מבוסס כרטיסים (Cards)

### Backend (Firebase)
- Firebase Authentication – אימות משתמשים
- Firestore – מסד נתונים בזמן אמת
- Firebase Storage – העלאת תמונות וקבצים
- Cloud Functions – לוגיקות אוטומטיות (אופציונלי)

יתרונות:
- עבודה בזמן אמת
- אבטחה גבוהה
- סקיילביליות אוטומטית
- תחזוקה מינימלית

## 📝 פוסטים ותגובות

### פוסטים
כל פוסט כולל:
- כותרת (חובה)
- תוכן (אופציונלי)
- תמונה (אופציונלי)
- קובץ מצורף (אופציונלי)
- מזהה יוצר
- שם משתמש היוצר
- תאריך יצירה

הרשאות:
- USER – עריכה / מחיקה של פוסטים אישיים
- ADMIN – עריכה / מחיקה של כל פוסט

### תגובות
- טעינה בזמן אמת (Realtime)
- משויכות לפוסט
- כוללות טקסט, שם משתמש ותאריך
- מחיקה ע"י כותב התגובה או ADMIN
- שמות משתמשים לחיצים → מעבר לפרופיל
